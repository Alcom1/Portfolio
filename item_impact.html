IMPACT

Get a real browser!

Controls

Aim: Point your cursor where you wish to fire.

Shoot: Left Mouse Button

Move: WASD

Role: All

Language: Javascript

Impact is a basic twin-stick style shooter demonstrating the use of collapsing n-gons in games. The game uses special collision detection between points and n-gons to create stationary objects that collapse, crater, and shrink when colliding with a projectile. The game features a sequence of levels that require all asteroids to be destroyed to advance.
