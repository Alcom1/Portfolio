IMPACT

Get a real browser!

Controls

Aim: Point your cursor where you wish to fire.

Shoot: Left Mouse Button

Move: WASD

Role: All

Language: Javascript

Impact is a basic twin-stick shooter demonstrating the use of interactive collapsing n-gons. The demo is an html canvas application written with a javascript module pattern. The demo uses a custom 2D vector library, point-polygon collision detection, and collision resolution that collapses the collided edge of an n-gon.
