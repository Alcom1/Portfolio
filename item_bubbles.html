BUBBLES

Role: All

Language: Processing

Bubbles is a graphics application designed for quickly drawing objects using lines and paths made of circles, and saving the image for printing. Bubbles uses a simple follow behavior to uniformly draw a path of circles. The application also has a responsive design to fit any window resolution, several options, and pdf exporting.
